Análisis estadístico de datos circulares

Datos vectoriales. Datos circulares

Una magnitud vectorial es aquella que, además de un valor numérico y su unidad (módulo) tiene una dirección y sentido, por ejemplo, la velocidad y dirección del viento. Esa dirección y sentido se analizarán como datos circulares. 

La llamada estadística circular o el análisis estadístico de datos circulares permite el tratamiento de datos angulares. Mientras que los datos lineales se presentan en la recta real, los datos circulares lo hacen en un círculo de radio unidad.

En el texto siguiente presentamos las principales medidas descriptivas para datos circulares y realización de gráficos.

Para realizar la práctica en RStudio será necesario:

  • Descargar el paquete de R denominado VecStatGraphs2D
  • Seguir el script denominado "script_vectorial.R" .
  • Descargar la carpeta de datos llamada vientos.
